Hierarquia de Seletores CSS

Abaixo estão os seletores básicos de CSS:

Selecionando um ID

Para selecionar uma tag pelo atributo ID utilize o # junto com o nome do ID.

Selecionando uma CLASS

Para selecionar uma tag pelo atributo CLASS utilize o .(ponto) junto com o nome da CLASS.

Selecionando pelo nome da TAG

Para selecionar uma tag basta escrever o nome da tag direto sem ponto ou sustenido.
